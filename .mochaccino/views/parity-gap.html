// KYBERAGENT DESKTOP — background analysis 2026-05-20
Ian's Electron app at ~/dev/kybernesis/kyberagent-desktop. Recently migrated from ChromaDB → sqlite-vec (same tech as arcana-provider-sqlite-vec). Has a working 10-step sleep pipeline, AI-assisted entity deduplication (Claude Haiku), and ARP scoping columns on facts (project_id, connection_id, classification). Schema shape misaligns with Arcana contracts on 5 axes: fact sourceType enum is different, no Memory table, entity edges only connect entities (not entity×memory), entity scopes live on facts not entities, and no surprisalScore column. The codebase is in high flux (TCP auth gate, SSE proxy, tunnel manager, connection registry all recently added). Not touching it now — adopt after it stabilises. Schema gap analysis documented for when Ian is ready.
